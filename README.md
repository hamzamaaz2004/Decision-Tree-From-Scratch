# Decision-Tree-From-Scratch
This repository contains the implementation of the Decision Tree algorithm in Machine Learning. The project demonstrates data preprocessing, model training, and evaluation using Python.

##  Project Overview
This project demonstrates the implementation of the Decision Tree algorithm for classification problems using Python.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Features
- Data preprocessing
- Model training
- Model evaluation
- Performance metrics

1. Clone the repository
2. Open the notebook
3. Run all cells
